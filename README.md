# cicd-hello-word
repo for just practicing cicd app deployments with full suite of testing for static and dynamic code scanning, image vuln scanning, and docker building and pushing before deployment to X location
uses enterprises standard tools like trivy and grype for scanning
uses sonarqube cloud for application scanning sonarcloud.io
connected to personal docker hub account
could have some kind of linting and formatting
if app is packaged via helm charts could do some kind of helm templating and helm building stage
